Data Format

If set, screenshots will first be sent to this OCR endpoint as a file upload (multipart/form-data with field file), and the returned text will be forwarded to your model API. Expected JSON response: { "text": "..." }.

The extension sends data in the following JSON format:

{
  "type": "selected_text|all_text|screenshot",
  "content": "...",
  "model": "selected-model-name",
  "pageUrl": "https://example.com",
  "pageTitle": "Page Title",
  "timestamp": "2024-01-01T12:00:00.000Z"
}

Usage

  • Add Profile: Enter profile name and server URL, click "Add Profile"
  • Fetch Models: Click to fetch available models from the selected server
  • Select Model: Choose the model to use from the dropdown
  • Send Selected Text: Right-click selected text and choose "Send Selected Text to API"
  • Send All Text: Right-click any page and select "Send All Visible Text to API"
  • Send Screenshot: Right-click any page and select "Send Screenshot to API"
